A weekly review club for Bitcoin Core PRs

All are welcome!

We'll host a series of weekly review clubs on the #bitcoin-core-pr-reviews Freenode IRC channel at 17:00 UTC on Wednesdays. All are welcome to join and ask questions!

Reviewing and testing PRs is the best way to start contributing to Bitcoin Core, but it's difficult to know where to start. There are hundreds of open PRs, many require a lot of contextual knowledge, and contributors and reviewers often use unfamiliar terminology. This weekly IRC club is for people who want to help review Bitcoin Core PRs but find the process intimidating. Everyone is welcome to speak and questions are encouraged!

To take part, you should:

  • clone the repo, checkout and build the PR branch and run all tests
  • review the code changes and read the comments on the PR
  • make a note of any questions you want to ask
  • join the IRC channel at 17:00 UTC on Wednesday

Other resources for new contributors

  • Read the Contributing to Bitcoin Core Guide. This will help you understand the process and some of the terminology we use in Bitcoin Core.
  • Look at the good first issues and up for grabs list.
  • Read the Bitcoin Core developer productivity tips.
  • Brush up on your C++. There are many primers and reference manuals available. Some contributors recommend C++ Primer Plus or Thinking in C++ which may be a little dated and doesn't cover C++11 or 14.
  • Blog posts on contributing to Bitcoin Core from Jimmy Song and John Newbery.
